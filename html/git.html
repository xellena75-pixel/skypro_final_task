Git

  • git init -инициализация нового репозитория в текущей дериктории.
  • git clone (url)-создание локальной копии удаленного репозитория.
  • git add (файл) - добавление изменений в индекс для последующего коммита .
  • git commit -m(сообщение)- создание нового коммита с указанным сообщением.
  • git status - просмотр состояния файлов в рабочей директории и индексе
  • git branch -управление ветками:создание ,удаление ,переименование
  • git checkout(ветка) - переключение между ветками
  • git puch - отправка изменений в удаленный репозиторий
